GenForgeAI Mobile UI Overhaul – Phase 1

Global Metroidvania/Fantasy Theme Foundation

Objective

Redesign the entire GenForgeAI Mobile application into a premium, animated 2D Metroidvania-inspired experience while preserving 100% of the existing functionality, navigation, providers, routing, API calls, state management, and business logic.

Do not remove or rename any routes, components, contexts, hooks, API endpoints, providers, or navigation structure.

---

Visual Direction

The application should feel like entering a magical forge where AI is creating worlds.

Style references:

- Dark Fantasy
- Metroidvania
- Hand-painted 2D environments
- Animated fantasy UI
- Modern premium mobile UX

Avoid making it look like a traditional productivity application.
